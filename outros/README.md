# my_cpp_lessons

Parâmetros a configurar no Visual Studio Code.

- Abrir os settings CTRL+SHIFT+P (Settings)
- Procurar por "Run inTerminal"
- Selecionar a caixa com a descrição: "Code Runner: Run in Terminal"
